Weather App Features

Home Features Installed
  • Responsive Navbar: The navbar includes a logo and navigation links for Home, Features, and Installed sections. The navbar is fully responsive, adapting to various screen sizes and ensuring a seamless user experience on both mobile and desktop devices. A theme toggle switch is available for users to switch between light and dark themes.
  • Theme Toggle Switch: Users can switch between light and dark themes using a modern flat-style toggle switch. In light theme mode, the toggle switch is green, and in dark theme mode, it changes to blue. The entire page, including the navbar and text, adjusts its color scheme based on the selected theme.
  • Search Functionality: The search bar allows users to enter the name of any city to fetch the current weather data. A search button is provided next to the input field for initiating the weather search.
  • Weather Information Display: Upon entering a valid city name, the app displays the current weather conditions including temperature, humidity, and wind speed. The temperature is shown in degrees Celsius (Â°C). Weather conditions are visually represented with corresponding icons (e.g., clouds, clear sky, rain, drizzle, mist, snow).
  • Error Handling: If an invalid city name is entered, an error message is displayed indicating that the city name is invalid.
  • Live Weather Updates: The app fetches live weather data from the OpenWeatherMap API, ensuring that users receive the most up-to-date information. Weather icons dynamically change based on the current weather conditions in the searched city.
