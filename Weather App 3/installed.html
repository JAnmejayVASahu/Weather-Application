Weather App Information

The Weather App is built using the following technologies and techniques:

  • HTML5: Used for structuring the content and layout of the web page.
  • CSS3: Employed for styling the elements and creating a visually appealing design.
  • Bootstrap 4: Integrated Bootstrap framework for responsive layout, components, and utilities.
  • JavaScript: Utilized JavaScript for dynamic functionality such as fetching weather data, error handling, and theme toggling.
  • OpenWeatherMap API: Accessed OpenWeatherMap API to retrieve live weather data based on user input.
  • Responsive Design: Implemented responsive design techniques to ensure optimal viewing experience across various devices and screen sizes.
  • Modern Flat-Style Toggle Switch: Used a modern flat-style toggle switch for toggling between light and dark themes, enhancing user experience.
